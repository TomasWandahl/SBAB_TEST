## Setup
### Backend
To run the Backend Server, install it locally using npm:

```
$ npm install
$ npm run dev
```

Copy the contents of .env.example to .env and enter your API_KEY

### Frontend
To run the Frontend Client, install it locally using npm:

```
$ cd client
$ npm install
$ npm run dev
```
